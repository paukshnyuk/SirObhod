# SirObhod
подготовка:
создать вк звонок, сохранить ссылку для подключения, выйти из звонка без его завершения
создать конфиг для ориг WG
скинуть на клиент
в [Interface] изменить\добавить MTU = 1280
в [Peer] изменить Endpoint на Endpoint = 127.0.0.1:9000
в приложении WG отредачить конфиг так, чтобы исключить термукс

на серваке:
закинуть server_vkp на сервак
дать права: chmod +x server_vkp
запуск: screen -S vkproxy
./server_vkp -listen 0.0.0.0:56000 -connect 127.0.0.1:портконфига

на клиенте(телефон):
в термуксе: termux-setup-storage
скопировать client_vkp на термукс
дать права: chmod +x client_vkp
запуск: termux-wake-lock
./client_vkp -peer айписервака:56000 -vk-link "ссылкаВК" -listen 127.0.0.1:9000
если обход не нужен выключить термукс в фоне: termux-wake-unlock
